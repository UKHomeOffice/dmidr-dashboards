## Run with Docker
```
$ docker compose up
```

## Run without Docker

### 1. Set up virtual env

```
$ python -m venv .venv
```

### 2. Start env

```
$ source .venv/bin/activate
```

### 3. Install requirements

```
$ pip install -r requirements.txt
```

### 4. Run app

```
$ python app/index.py
```
